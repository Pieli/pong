# Dependencies

Debian based systems:
```
$ sudo apt install libsdl2-dev
```

Arch based systems:
```
$ sudo pacman -S sdl2
```

```

```

# Run animation
change permission:
```
$ chmod +x pong
```
run:
```
$ ./pong
```
